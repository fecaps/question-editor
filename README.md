# Question Editor App

### Pre requisites:

- [Docker](https://docs.docker.com/install/)
- [Docker Compose](https://docs.docker.com/compose/install/)

### Resources

- [Redis Commander](http://localhost:8081)
- [Web API](http://localhost)
- [Web APP](http://localhost:3000)

### Usage

- Build:

```bash
$ docker-compose up -d --build
```

- Setup:

```bash
$ docker-compose up -d
```

- Stop:

```bash
$ docker-compose down
```
